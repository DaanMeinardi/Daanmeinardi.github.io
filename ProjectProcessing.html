Processing

Processing is a Java-based coding platform to create small games and animations. I started learning this in the first module of the study and am still doing new things with it every week. For one project the goal was to create an animation of a creature. With much freedom, I chose to make a frog. It can jump around and catch flies as you can see in the following GIF.

You can move the frog by pressing the "a" and "d" keys and jump with spacebar. To generate the new flies you can click on the screen and it will spawn a new fly.

Creature I made in module 1 as programming assignment.
